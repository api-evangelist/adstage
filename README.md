# AdStageThis is a repo for managing machine readable artifacts for the [AdStage APIs](https://www.adstage.io). The goal is to maintain and evolve an OpenAPI, Postman Collection, and APIs.json index for the [AdStage APIs](https://www.adstage.io).They are a constant work in progress, and will be updated on a regular basis when new information is added, or more of the specifications are supported.Feel free to submit a pull request to evolve any of the artifacts, and leave GitHub issues if you have any questions, comments, or feedback.- [Website](https://www.adstage.io)- [Developer](https://www.adstage.io)- [Twitter](https://twitter.com/adstage)The [APIs.json](https://github.com/api-evangelist/adstage/blob/master/apis.json) index provides links to all the relevant links for the AdStage APIs, including the OpenAPI and Postman if they are present.
